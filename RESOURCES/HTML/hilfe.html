Benutzerverwaltung

Benutzer anlegen

  1. Beim ersten Start muss ein Benutzer angelegt werden damit das Programm genutzt werden kann.
    Diesem Benutzer werden automatisch Adminrechte zugewiesen.
  2. Jeder angemeldete Benutzer, egal ob normaler Benutzer oder Aministrator kann weitere Benutzer anlegen.

Benutzerrechte ändern

  1. Die Rechte vorhandender Benutzer können nur durch Benutzer mit Admin-Rechten geändert werden.
    So kann also nur ein Benutzer mit Admin-Rechten anderen Benutzern Admin-Rechte gewähren oder
    diese entziehen.
  2. Es muss mindestens ein Administrator in der Datenbank stehen.
    Steht nur ein Administrator in der Datenbank, ist es nicht möglich, diesem die Adminrechte zu entziehen,
    bevor nicht einem anderen Benutzer Adminrechte gewährt werden.

Benutzer löschen

  1. Vorhandene Benutzer können nur durch Benutzer mit Admin-Rechten gelöscht werden.



Besucher Anmeldung

Besucher Anmeldung (Bereits gespeicherter Besucher)

  1. Anstatt die Ausweisnummer des besuchers jedes Mal von Hand eingeben zu müssen, können Sie auch den Besucher aus der Liste der Besucher auswählen und anschließend auf den Button "Besucher anmelden" klicken. Drücken Sie nun die Tastenkombination Strg + V, woraufhin die Ausweisnummer des zuvor gewählten Besuchers automatisch eingefügt wird.


Besucher Anmeldung (Neuer Besucher)

  1. Wenn Sie eine Besucher-Anmeldung für einen noch nicht gespeicherten Besucher ausfüllen wollen, klicken Sie auf den Button "Besucher anmelden" und geben Sie die Ausweisnummer des Besuchers ein. Im darauf erscheinenden Formular füllen Sie alle grün markierten Eingabefelder aus um die Besucherdaten zu speichern. Anschließend öffnet sich ein Formular zur Eingabe der Anmeldedaten.



Chipkarten Ausgabe

Chipkarten-Ausgabe (Bereits gespeicherter Besucher)

  1. Anstatt die Ausweisnummer des Besuchers jedes Mal von Hand eingeben zu müssen, können Sie auch den Besucher aus der Liste der Besucher auswählen und anschließend auf den Button "Chipkarten Ausgabe" klicken. Drücken Sie nun die Tastenkombination Strg + V, woraufhin die Ausweisnummer des zuvor gewählten Besuchers automatisch eingefügt wird.


Chipkarten-Ausgabe (Neuer Besucher)

  1. Wenn Sie eine Chipkarte an einen noch nicht gespeicherten Besucher ausgeben wollen, klicken Sie auf den Button "Chipkarten Ausgabe" und geben Sie die Ausweisnummer des Besuchers ein. Im darauf erscheinenden Formular füllen Sie alle grün markierten Eingabefelder aus um die Besucherdaten zu speichern. Anschließend öffnet sich ein Formular zur Eingabe der Chipkartendaten.



Datenschutz-Grundverordnung (DSGVO)

Automatische Löschung personenbezogener Daten

Personenbezogene Daten sollen laut DSGVO nach einer gewissen Zeit gelöscht werden.
Jedesmal wenn das Programm geöffnet oder eine Bedienerummeldung stattfindet, wird folgendes ausgeführt.

  1. Besucheranmeldungen bei denen das Datum im Feld "Anmeldung Bis" älter als der im Feld "Löschzeit in Tagen" angegebene Wert ist, werden automatisch gelöscht.
  2. Chipkartenausgaben bei denen das Datum im Feld "Objekt betreten" älter als der im Feld "Löschzeit in Tagen" angegebene Wert ist, werden automatisch gelöscht.
  3. Besucherdaten werden gelöscht, wenn das Registrierungsdatum älter als der im Feld "Löschzeit in Tagen" angegebene Wert ist und wenn dem Besucher weder Besucheranmeldungen noch Chipkartenausgaben zugeordnet sind.
Weitere Informationen gemäß Artikel 13 und 14 DSGVO finden Sie im Menü unter "Hilfe / Datenschutz-Grundverordnung"
